# Archive

This folder keeps material that should not sit in the GitHub Pages root, but may still be useful as reference.

## Categories

- `business-notes/` — product notes, PRDs, research notes, and working documents.
- `design-files/` — editable design files and font/design artifacts.
- `external/` — copied third-party or external source trees.
- `internal/` — local agent notes and scratch files that are not public entry points.
- `legacy-docs/` — documentation for retired single-file prototypes.
- `legacy-pages/` — old demos, prototypes, or pages no longer linked from `launch.html`.
- `presentations/` — slide decks and generated presentation sources.
- `reference-dumps/` — large source/reference dumps used to derive current diagrams or patterns.

## Rule

New public Pages entry points should be linked from `launch.html` or documented in the root `README.md`.
Everything else should either live in an active module directory or be moved here.
